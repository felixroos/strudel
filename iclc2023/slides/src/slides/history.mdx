# The journey

![ports](./img/ports.png)

- (2009) TidalCycles: popular live coding tool for music created by Alex McLean
- (2021) Vortex: port of TidalCycles from haskell to python
- (2022) Strudel: initially ported from Vortex to JavaScript, now has a life of its own
- first commit by Alex McLean on Jan 22, 2022
- Alex ported tidal's core pattern representation from Vortex
- I joined the project on Feb 4, 2022 creating the REPL + audio engine + many APIs
- Today: 25 contributors, ~2,900 commits, 11 Releases
- Version 1.0 will be released on Jan 22, 2024 (hopefully)
